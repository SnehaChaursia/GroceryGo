## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/SnehaChaursia/GroceryGo.git
   ```
2. Navigate to the project directory:
   ```bash
   cd GroceryGo
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```
